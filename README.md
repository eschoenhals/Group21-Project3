# Group21-Project3

Convolutional Neural Network to classify whether images contain the official University of Arkansas Razorback logo for trademark enforcement.

## Authors

- Emmett Schoenhals
- Kailey Phillips

## Files

- `project3-dasc-41103.ipynb`- Notebook containing dataset development, image preprocessing, CNN architecture design, model training, and performance evaluation. 
- `Group_21_CNN_FullModel.ph` - Trained CNN model for predicting official logo presence (1) or absence (0).
- `project 3 photo.zip` - Custom dataset of 97 images collected from Etsy, organized into official_logo/ and not_official_logo/ subdirectories.
